
# Spring Boot Course Management System

A REST API project developed using Spring Boot and MySQL.

## Features
- Student CRUD Operations
- Course CRUD Operations
- Many-to-Many Mapping
- Validation using Jakarta Validation
- Global Exception Handling
- Swagger API Documentation

## Technologies Used
- Java
- Spring Boot
- MySQL
- Hibernate / JPA
- Swagger OpenAPI
- Maven

## API Testing
Swagger UI was integrated for testing REST APIs.
